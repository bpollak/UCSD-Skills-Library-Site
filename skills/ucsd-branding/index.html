UC San Diego branding

Authoritative values live in references/brand-spec.md (verified against brand.ucsd.edu). Read it before applying anything. If asked about an edge case the spec doesn't cover, check brand.ucsd.edu rather than guessing — hex codes and usage rules are strict.

Core rules (always apply)

  1. UC San Diego Blue #00629B is the required, dominant color. Buttons, links, focus rings, primary chart series. Accents (Turquoise, Magenta, Green, Orange, Citron) stay secondary — never dominant.
  2. Navy #182B49 for text and mastheads, not black. Headers/wordmark on navy with a Gold #C69214 rule is the recognizable campus web motif.
  3. Yellow #FFCD00 sparingly — active/highlight accents, or as the primary action color on navy/dark surfaces (the brand's highest-contrast pairing).
  4. Typography: Brix Sans is the licensed primary — do NOT bundle it without a license. Use Roboto (the designated free substitute, Google Fonts), falling back to Helvetica Neue/Helvetica/Arial.
  5. Accessibility: WCAG AA — text contrast ≥ 4.5:1. UCSD Blue and Navy pass on white; Yellow/Citron NEVER as text on white; darken Cool Gray to ≈#5D6066 for small muted text.
  6. Logos are licensed assets. Never fabricate or trace the campus logo; use a text wordmark ("UC San Diego" bold + unit name) until the official SVG is downloaded from brand.ucsd.edu and dropped into the project.

Applying to a Tailwind v4 / shadcn app

Map theme tokens in globals.css (full mapping with exact values in the spec's "shadcn token mapping" section):

  • --primary → UCSD Blue, --foreground → Navy, --secondary/--muted → Sand-tinted neutrals, --border → stone-tinted hairline, --ring → Blue
  • Sidebar/masthead tokens → Navy background, light foreground, Yellow active accent, gold border rule
  • Charts → Blue, Turquoise, Gold, Magenta, Green (in that order)
  • Dark mode → navy-anchored background (#0E1A30), Yellow as --primary with Navy foreground
  • Define the raw palette as --ucsd-* CSS variables so components can reference brand colors directly (e.g. border-(--ucsd-gold))

Page chrome — match Decorator 5, not an invented motif

Ground truth source: https://developer.ucsd.edu/design/v5-kitchen-sink/ and https://cdn.ucsd.edu/cms/decorator-5/styles/base.min.css (verified June 2026 by reading the actual source CSS).

Verified Decorator 5 chrome stack (top to bottom)

1. .layout-header wrapper

  • background-color: #2b92b9 (medium blue strip behind the white title bar)
  • Contains .layout-title and optionally .layout-login

2. .layout-title — the white site-name band

  • background: #fff, height: 92px, padding: 1.5em 0
  • font-family: Roboto, sans-serif
  • Left: .title-header link — color: #000, font-size: 1.35rem, letter-spacing: 1px, text-transform: uppercase, float: left
    • Hover/focus: color: #666
    • Two variants: .title-header-large (desktop, margin-top: 5px) and .title-header-short (mobile abbreviation, display: none on desktop)
  • Right: .title-logo — image-replaced sprite link to ucsd.edu, width: 229px; height: 65px, float: right, text-indent: -999em, background from sprite_base.png at position 0 -3px
    • Hidden below ~560px (mobile shows a small logo in the navbar-header instead)

3. .navbar / .navbar-default — the UCSD Blue navigation bar

  • background-color: #00629b, min-height: 50px, border-bottom: none
  • Nav links: color: #fff, padding: 9px 15px
  • Hover/active/open state: background-color: #004268 (darker blue)
  • Gold active-tab underline: border-bottom: 3px solid #ffcd00 on .active > a
  • Search toggle button on far right
  • Mobile: hamburger toggle (background-color: #00629b, white icon-bars), UCSD logo image shown in navbar-header on xs screens only

4. .layout-main — width: 100%, no background (inherits page bg)

5. .footer / .layout-footer — the UCSD Blue footer

  • background-color: #00629b, color: #fff, font-size: 90%
  • border-top: 1px solid #ccc
  • Left column (.col-sm-8): address line + copyright + .footer-links list
    • Footer links: color: #fff, text-decoration: underline, pipe-separated via border-right: 1px solid #fff
  • Right column (.col-sm-4): .footer-logo image (width: 158px; height: 30px; float: right) — white UCSD wordmark logo from https://cdn.ucsd.edu/developer/decorator/5.0.2/img/ucsd-footer-logo-white.png
  • Optional .prefooter band above footer: background-color: #f1f1f2, color: #182b49, used for supplemental nav/contact links

Key corrections vs. prior skill version

  • The gold rule is NOT a structural header border. It appears only as the border-bottom: 3px solid #ffcd00 on the active nav tab in the blue navbar. There is no gold rule at the bottom of the white .layout-title band.
  • The footer is UCSD Blue #00629b, not navy #182b49.
  • The .layout-header outer wrapper is #2b92b9 (medium blue), not white or navy.
  • Teko SemiBold is used for h1/h2 headings (color: #00629b for h1, color: #182b49 for h2), loaded from cdn.ucsd.edu/cms/decorator-5/styles/teko.css. Body and UI text use Roboto. Keep Teko surgical — headings only.
  • .layout-container: max-width: 1200px (capped at 960px below 1200px viewport), width: 98%, margin: 0 auto.

Logo assets (from CDN — do not redraw)

  • Header sprite: https://cdn.ucsd.edu/cms/decorator-5/styles/img/sprite_base.png (and sprite_base2x.png for retina)
  • Footer wordmark: https://cdn.ucsd.edu/developer/decorator/5.0.2/img/ucsd-footer-logo-white.png

Verify after applying

Check the four highest-visibility surfaces: sign-in page, any public-facing page, the app shell/sidebar, and one email template. Confirm: blue (#00629b) dominates, black site title text in white band, gold underline on active nav tab only, no yellow-on-white text, footer is blue (not navy), focus rings visible.
