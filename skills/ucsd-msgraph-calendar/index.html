Microsoft 365 Calendar Lookup

Check UC San Diego Microsoft 365 calendars for meetings, availability, agendas, and schedule context.

This skill is still being reviewed and may change.

Skill ID

ucsd-msgraph-calendar

Best For

Calendar-enabled agents

Source

View on GitHub

Enable in TritonAI Harness

This skill is available in the TritonAI Harness. If it is marked as generally available, you can enable it by toggling it on in the Harness interface using the Skill ID above.

View full skill on GitHub

Description: Query Microsoft Exchange/365 calendars via Microsoft Graph API using OAuth2 device code flow. Use when a user asks about their schedule, meetings, or availability. Trigger on: "my calendar", "what meetings", "today's schedule", "what am I doing", "check calendar", "am I free", "meeting with", "calendar", "schedule", "agenda", "this afternoon", "this week", "appointment".
